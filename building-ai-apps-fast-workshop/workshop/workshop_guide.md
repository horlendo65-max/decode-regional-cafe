# 90-minute workshop guide

## Mission

Use AI coding agents, reusable skills and workflows to turn a prepared scaffold into a deployed AI Expense Advisor MVP.

This is a practical workshop. Participants should leave with a GitHub repository, a live deployed URL and a short demo story.

## Timeline

| Minutes | Activity | Output |
|---:|---|---|
| 0–10 | Project story and architecture | Everyone understands the MVP |
| 10–20 | Local setup | API and web run locally |
| 20–35 | Gemini configuration | First AI request works |
| 35–52 | Agent-assisted workflow | Expense workflow completed |
| 52–65 | M-PESA analysis | Useful structured response |
| 65–78 | Deployment | Vercel projects configured |
| 78–90 | Demo and extensions | Live URL + lessons learned |
