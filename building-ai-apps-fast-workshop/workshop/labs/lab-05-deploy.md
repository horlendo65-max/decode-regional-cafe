# Lab 05 — Deploy to Vercel

## Goal

Deploy both API and web app.

## Steps

1. Push your repository to GitHub.
2. Deploy `starter/api` to Vercel.
3. Add `GEMINI_API_KEY` to API environment variables.
4. Test `/health`.
5. Deploy `starter/web` to Vercel.
6. Set `NEXT_PUBLIC_API_URL` to the deployed API URL.
7. Test the full app.

## Checkpoint

You have a live deployed URL that can analyse pasted M-PESA text.
