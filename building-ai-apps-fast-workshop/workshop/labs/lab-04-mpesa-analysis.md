# Lab 04 — M-PESA Statement Analysis

## Goal

Test real product value by analysing copied statement text.

## Sample text

```text
NAIVAS SUPERMARKET KES 2400
UBER TRIP KES 680
JAVA HOUSE KES 1250
SAFARICOM AIRTIME KES 500
KPLC TOKEN KES 1500
```

## Prompt improvement

Ask the coding agent to improve the output so it includes:

- spending categories;
- summary;
- three savings ideas;
- safe financial disclaimer;
- Markdown formatting.

## Checkpoint

The response is specific, structured and useful.
