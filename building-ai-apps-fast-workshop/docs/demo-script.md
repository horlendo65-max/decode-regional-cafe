# Demo Script

## Demo title

AI Expense Advisor: From M-PESA Text to Budget Recommendations

## Demo setup

Open:

- deployed frontend URL;
- repository on GitHub;
- `ExpenseAnalysisWorkflow` file;
- `ExpenseAdvisorAgent` file.

## Sample input

```text
NAIVAS SUPERMARKET KES 2400
UBER TRIP KES 680
JAVA HOUSE KES 1250
SAFARICOM AIRTIME KES 500
M-PESA SEND MONEY KES 3000
KPLC TOKEN KES 1500
```

## Demo steps

1. Paste the sample statement.
2. Ask: "Analyse my spending and suggest how I can save more this month."
3. Show the response.
4. Highlight categories and recommendations.
5. Open the workflow file and explain the orchestration.
6. Open the skills folder and explain reuse.
7. Mention how the same architecture can become Health or Agriculture advisor.
8. Show the deployed URL.

## Demo success statement

"In under 90 minutes we moved from idea to a deployed AI MVP by composing agents, skills and workflows instead of coding everything from scratch."
