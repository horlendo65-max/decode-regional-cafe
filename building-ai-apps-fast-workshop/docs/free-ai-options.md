# Free AI options

## Runtime model
Use the Gemini API free tier where available. Free-tier limits, model names and regional availability can change; check Google AI Studio during setup. The starter defaults to `gemini-2.5-flash`, configurable with `GEMINI_MODEL`.

## Coding agents
- GitHub Copilot Free: useful inside supported editors for explaining and editing the scaffold.
- Gemini Code Assist / Gemini-powered editors: useful as an alternative coding agent.

A coding-agent subscription or free allowance does not automatically make that assistant a public inference API for the deployed application.
