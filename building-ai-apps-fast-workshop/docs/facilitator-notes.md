# Facilitator notes

Have the reference API and web app deployed before the session. Ask participants to create Gemini keys and Vercel accounts beforehand. Pair mixed-experience attendees. The core checkpoint is workflow composition by minute 55; customization is secondary.

Avoid live PDF parsing in the core lab: statement layouts vary and sensitive data handling deserves more time. Use sanitized pasted text. Daraja is a demonstration extension because callback setup can consume the deployment window.
