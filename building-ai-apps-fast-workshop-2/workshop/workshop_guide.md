# 90-minute workshop guide

## Mission
Use an AI coding agent to turn the prepared scaffold into a deployed AI expense advisor. You are not starting from a blank folder: agents, skills, workflows, prompts and deployment boundaries are prepared so you can focus on composition and iteration.

## Timeline
| Minutes | Activity |
|---:|---|
| 0–10 | Idea → agent → skills → workflow architecture |
| 10–20 | Fork/clone, install and inspect the scaffold |
| 20–35 | Configure Gemini and test the Express health endpoint |
| 35–55 | Use the coding-agent prompt to assemble the expense workflow |
| 55–68 | Run M-PESA analysis and improve the prompt/UX |
| 68–82 | Push to GitHub and deploy API + web to Vercel |
| 82–90 | Demo and optional Health/Agri/Daraja extension |
